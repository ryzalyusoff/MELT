Exam Create an Exam Open the Application. Click on the ‘EXAM’ button. On the left panel towards the bottom, click on the ‘Add an Exam’ button. Enter any instructions you may have for the examination (These will be shown to the student when carrying out his/her exam) and click the ‘OK’ button. You will be taken back to the Exam Overview section where you can see your exam added (With the last exam added at the bottom of the list on the left panel). Activate an Exam Open the Application. Click on the ‘EXAM’ button. Select the exam you would like to activate from the left panel using the checkbox and then click the ‘Activate’ button. You will then see an ‘Activated’ next to the selected exam. Add a Section Open the Application. Click on the ‘EXAM’ button. Select the exam you would like to add the section to from the left panel using the checkbox and then click the ‘Edit’ button. On the left panel towards the bottom, click on the ‘Add an Section’ button. Enter the following information An appropriate section name. The time limit that will be assigned to that particular section. The number of subsections that you would like to include in this section. Click the ‘OK’ button. Delete a Section Open the Application. Click on the ‘EXAM’ button. Select the exam you would like to add the section to from the left panel using the checkbox and then click the ‘Edit’ button. Select the section you would like to delete from the left panel using the checkbox and then click the ‘Delete’ button. Change the time limit Open the Application. Click on the ‘EXAM’ button. Select the exam you would like to add the section to from the left panel using the checkbox and then click the ‘Edit’ button. Select the section you would like to edit from the left panel using the checkbox and then click the ‘Edit’ button. Change the time limit from the top of the page. Click on the ‘Update Section’ button to save your changes. Add a Subsection to a Section Open the Application. Click on the ‘EXAM’ button. Select the exam you would like to add the section to from the left panel using the checkbox and then click the ‘Edit’ button. Select the section you would like to edit from the left panel using the checkbox and then click the ‘Edit’ button. Click the ‘Add a Subsection’ button next to the time limit. Click on the ‘Update Section’ button to save your changes. Add a Question (not to Subsection) Open the Application. Click on the ‘EXAM’ button. Select the exam you would like to add the section to from the left panel using the checkbox and then click the ‘Edit’ button. Select the section you would like to edit from the left panel using the checkbox and then click the ‘Edit’ button. Click the ‘Add a Question’ button next to the ‘Add a Subsection’ button. A table will appear with the available questions. Select the question that you wish to add to the section. Click on the ‘Add Question’ button at the bottom of the page. Click on the ‘Update Section’ button to save your changes. Add a Question from a Subsection Open the Application. Click on the ‘EXAM’ button. Select the exam you would like to add the section to from the left panel using the checkbox and then click the ‘Edit’ button. Select the section you would like to edit from the left panel using the checkbox and then click the ‘Edit’ button. Click the ‘Add a Question’ button which is next to the subsection that you wish to add a question to. A table will appear with the available questions. Select the question that you wish to add to the subsection. Click on the ‘Add Question’ button at the bottom of the page. Click on the ‘Update Section’ button to save your changes. Delete a Question from a Section Open the Application. Click on the ‘EXAM’ button. Select the exam you would like to add the section to from the left panel using the checkbox and then click the ‘Edit’ button. Select the section you would like to edit from the left panel using the checkbox and then click the ‘Edit’ button. Click the ‘Delete’ button which is next to the question that you wish to delete. Click on the ‘Update Section’ button to save your changes. Delete a Subsection Open the Application. Click on the ‘EXAM’ button. Select the exam you would like to add the section to from the left panel using the checkbox and then click the ‘Edit’ button. Select the section you would like to edit from the left panel using the checkbox and then click the ‘Edit’ button. Click the ‘Delete’ button which is next to the subsection that you wish to delete. Click on the ‘Update Section’ button to save your changes. Question Add a Multiple Choice Question (MCQ) Open the Application. Click on the ‘QUESTION’ button. Enter the Question text. Under ‘Answer Text’, enter the answer text options Select which answer/s is correct from beneath ‘Correct Answer’ by selecting the checkboxes. Click the ‘Add’ button below. You question will appear in the table on the right side of the window. Edit a Question Open the Application. Click on the ‘QUESTION’ button. Select a question from the list on the right side of the window (which will then be highlighted). Click the ‘Edit’ button beneath the list. Here you have multiple options which are as follows: Change the question text from the top field. Change which is the correct answer/s from the checkboxes below ‘Correct Answer’. Change the answer text options by modifying the text fields below ‘Answer Text’. Add possible answers (provided you have less than 6) by adding text in the text fields below ‘Answer Text’. Deleting answer text options by removing the corresponding text from the text fields below ‘Answer Text’, and also unchecking the checkboxes below ‘Correct Answer’ if they are selected. Once you have completed this step, click the ‘Update’ button to save your changes. Delete a Question Open the Application. Click on the ‘QUESTION’ button. Select a question from the list on the right side of the window (which will then be highlighted). Click the ‘Delete’ button beneath the list.
